View summary according to following models:-

Glove

Model will find sentence vector of query and find relevance using cosine similarity or euclidean distance.Then returns summary according to text rank algorithm.

FastText

Model will find sentence vector of query and find relevance using cosine similarity or euclidean distance.Then returns summary according to text rank algorithm.

Model will find sentence vector of query and find relevance using cosine similarity or euclidean distance.Then according to text rank algorithm top sentences which are then paraphrased to get summary.

Model will cluster document sentences according to similarity and return the cluster closest to the query.
